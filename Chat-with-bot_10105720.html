MVP CLL Phase 1 : Chat with bot

Created by Mai Ngoc Huyen, last modified on Nov 24, 2023

In the page detail bot, click button Chat with bot which is on the right corner of page, system show popup chat with bot. User can chat with CHL Assistant bot or Specialized bot that user bought from marketplace or Customer bot that user created.

  • Bot list : Contain all Bot include: CHL assistant, Specialize bot they buy from the market place, Specialized bot they buy free from the marketplace, all personal bot user created. If user change the bot the name and avatar would change also. The default bot is CHL assistant
  • Bot name : Display the selected bot name
  • Bot avatar : Display the selected bot avatar
  • Message history : Display list of message content that user has sent/received. Display default maximum 20 sent messages (including Text/Image/Video/Icon)Â  whenever user clicks to view pop-up. User scrolls up, system will load previous 20 messages. If there aren't any messages before, system just displays empty chat box
  • Maximum length for typing message to AI : 1500 characters
  • User click button Send to send message
  • User click button Close to close the popup
