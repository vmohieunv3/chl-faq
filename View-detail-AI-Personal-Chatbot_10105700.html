MVP CLL Phase 1 : View detail AI Personal Chatbot

Created by Mai Ngoc Huyen, last modified by Dương Thị Cúc on Nov 29, 2023

In the Profile page, click tab “AI Assistant” to view detail bot

  • Number of free specialized bot : Hover the icon to show content of tooltip “This is the number of free specialized bot provide based on tiers” 
  • Number of personal bot : Hover the icon to show content of tooltip “This is the number of bot you can create (obtain via marketplace, tier) / number of bot you can create”
  • Message : Hover the icon to show content of tooltip “This is the number of message you can obtain (Via marketplace, tier) / number of message you have based on your tier”
  • Bot name: This field is a dropdown list. User click to select a value in the list. Values in this dropdown list are the bots that User already created.
  • Bot status : Display when “Bot name” is selected. This field is auto filled by system. The value can be “Enable” or “Not enable”. The status will change to “Enable” when Ingest data status is “Complete” 
  • Ingest data status : Display when “Bot name” is selected. This field is auto filled by system. The value can be “Failed” or “Processing” or “Complete” depending on the status of the bot reading data 
  • Error log : Display when “Bot name” is selected. This field shows error log during reading data inputted. 
  • User click “View all” button, the system displays a popup showing error detail list. 
  • User click “Go to marketplace” , the system redirect to marketplace page
