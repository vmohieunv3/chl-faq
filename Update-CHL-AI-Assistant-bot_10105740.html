MVP CLL Phase 1 : Update CHL AI Assistant bot

Created by Mai Ngoc Huyen, last modified on Nov 24, 2023

User click “Update” button, the system display popup update bot

  • Name : This field can not be editable, the default value for this field is CHL AI Assistant
  • Role : This field can not be editable, the default value for this field is Customer support
  • Language : This field can not be editable, the default value for this field is English
  • Upload PDF : This field is required. The new file data will overwrite the old file. Rule for uploading PDF :  Terms for uploading the PDF file
  • “Cancel” button : User click this button to close the popup, back to detail page of AI CHL Assistant
  • “Save” button : This button is only enabled when at least 1 PDF file is uploaded. Click this button to update the bot successfully.
