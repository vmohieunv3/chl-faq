MVP CLL Phase 1 : Update Specialized Bot

Created by Mai Ngoc Huyen on Nov 24, 2023

In page detail bot, user click “Update” button, the system display popup update bot

  • Bot avatar : This field is required. Users need to upload the picture with maximum size is 10MB and supported format is .JPG, .JPEG, .PNG, .JPE, .HEIC. After user chose picture, system display this picture. User can click button “Clear the current one” to delete current avatar. If user delete current avatar, display button “Upload a new picture”
  • Bot name : This field is text input and required. Maximum length for this field is 100 characters and no special characters are allowed. 
  • Role : This field can not be editable. The default value for this field is Customer support
  • Language : This field can not be editable. The default value for this field is English
  • URL : This field is text input and optional. User can input an URL link.. Rule for uploading URL link : Terms for uploading the URL link
  • All relevant pages : This field is a checkbox and default value is untick. User is not required to tick this field. When user tick on this checkbox, the system will crawl all data related to the inputted URL 
  • Upload PDF : This field is required. The new file data will overwrite the old file. Rule for uploading PDF : Terms for uploading the PDF file. 
  • “Save” button : This button is only enabled when at least 1 PDF file is uploaded . When user click this button, system update the bot successfully. 
  • “Cancel” button : User click this button to close popup and back to detail bot page
