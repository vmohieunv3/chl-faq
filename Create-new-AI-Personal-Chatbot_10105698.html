MVP CLL Phase 1 : Create new AI Personal Chatbot

Created by Mai Ngoc Huyen, last modified by Dương Thị Cúc on Nov 29, 2023

In the page View detail bot, click button “Create new ChatBot”, system display the popup Create new Chatbot

  • Bot avatar :  Click button “Upload an avatar” to open the library from local device. This field is required. Users need to upload the picture with maximum size is 10MB and supported format is .JPG, .JPEG, .PNG, .JPE, .HEIC. After user chose picture, system display this picture. User can click button “Clear the current one” to delete current avatar. If user delete current avatar, display button “Upload a new picture”
  • Name : This field is text input and required. Maximum length for this field is 100 characters and no special characters are allowed. 
  • Role : This required field is dropdown list. User choose 1 value in the list. The values in list are : Customer support, Assistant
  • Language : This field can not be editable. The default value for this field is English
  • Domain embed : Format is https:xxxxx. Maximum length for this field is 100 characters
  • URL : This required field is text input. User can input an URL link. Rule for uploading URL : Terms for uploading the URL link
  • All relevant pages : This field is a checkbox and default value is untick. User is not required to tick this field. When user tick on this checkbox, the system will crawl all data related to the inputted URL 
  • Upload PDF : This field is required. The new file data will overwrite the old file. Rule for uploading PDF : Terms for uploading the PDF file. 
  • “Save” button : This button is only enable when required field is filled. When user click this button, system create new bot successfully and redirect to AI Assistant tab
  • “Cancel” button : User click this button to close popup and back to AI Assistant tab
