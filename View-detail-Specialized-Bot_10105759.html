MVP CLL Phase 1 : View detail Specialized Bot

Created by Mai Ngoc Huyen on Nov 24, 2023
  • Bot name: This field is a dropdown list. User click to select a value in the list. Values in this dropdown list are the bots that Admin already created.
  • Bot status” : Display when “Bot name” is selected. This field is auto filled by system. The value can be “Enable” or “Not enable”. The status will change to “Enable” when Ingest data status is “Complete” 
  • Ingest data status : Display when “Bot name” is selected. This field is auto filled by system. The value can be “Failed” or “Processing” or “Complete” depending on the status of the bot reading data 
  • Error log : Display when “Bot name” is selected. This field show error log during reading data inputted. 
  • User click “View all” button, the system displays a popup showing error detail list. 
  • User click “Update” button, the system display popup update bot
